## Задача

За последние 24 часа найди значимые сигналы из медленных источников по фокус-областям ниже. Используй WebSearch (по 1-2 запроса на область). Дату бери из системного контекста (поле ISO выше) — далее обозначена как <ДАТА>.

## Фокус-области пилота

- **AI/LLM** — новые модели, агенты, инструменты для управленца и разработчика.
- **Инженерия 1С/PostgreSQL** — релизы, практики (профессиональный домен пилота, PACK-1c-platform).
- **Рынок труда / сокращения 2026** — сигналы по найму в IT, диверсификация дохода (релевантно карьерному переходу и кэш-машинке).
- **IWE/платформа** — значимые апдейты инструментов экзокортекса (по желанию, если всплывёт).

## Шаги

1. По каждой области — WebSearch, отбери максимум 3 значимых материала с ссылкой.
2. Классифицируй по приоритету: **P0** (требует действия <24ч), **P1** (к сведению), **P2** (фон).
3. Шум отбрось. Каждый сигнал — одна строка: `[P?] краткое описание | ссылка | действие`.

## Запись результата

Запиши файл: `{{WORKSPACE_DIR}}/{{GOVERNANCE_REPO}}/inbox/agent/scout/<ДАТА>.md`

Структура:

```
---
date: <ДАТА>
generated_by: scout-daily
findings_count:
  P0: N
  P1: M
  P2: K
---

# Scout <ДАТА>

## AI/LLM
- [P?] ... | <url> | действие: ...

## Инженерия 1С/PostgreSQL
- [P?] ...

## Рынок труда / доход
- [P?] ...

## IWE/платформа
- [P?] ...
```

Если значимых находок нет в области — напиши «- значимого нет». Если по всем 0 — всё равно создай файл с `findings_count` по нулям (Day Open проверяет наличие файла).
